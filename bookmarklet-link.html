New Collect JIRA Bookmarklet

Below link is bookmarklet link that you should add/drag to bookmarks in order to install it as bookmarklet. Prerequisite is that this bookmarklet must be run on while you are on JIRA page. When run, dialog will pop-up, asking for start date of the worklogs you wish to export. Expected format is provided as default value in the prompt dialog, and this is "yyyy-mm-dd" (i.e. "2016-11-22", entered wihtout quotes, of course). After providing start date, another prompt dialog will popup asking for end date. Same rules apply, for second dialog as for the first one. After both dates are provided, dialog should popup with exported data in format that is expected from previously used Submit DR bookmarklet. Copy this exported text and paste it in the dialog that pops up from Submit DR dialog. You already know all this :D. Have fun!
